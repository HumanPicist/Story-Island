# 操作流水

> **倒序：最新在最上。** 新条目插入在本说明下方、列表顶部（不要追加到文件末尾）。
> 每次 init / 推进 / 干预 / 展开 / 盘点 / 编年 / 日结 记一行。
> 格式：`现实时间 · 动作 · 世界内时间 · 产物 · 骰点/备注`
>
> **限制·读取规约**：本文件**只保留最新 10 条**；更早的全部在 [[_log-归档]]（只增不删的历史兜底，满 10 条时由盘点自动建并溢出过去）。
> 写新条目 / 冷启动时**只读最新几条即可**，回溯久远操作才去翻归档。
>
> **长度规约**：一条 ≤ 约 2 行。**产物**段只列「动到的对象/文件 + 一句最关键事件」；逐时叙事 / 全部骰值 / 记忆·数据层逐条细节归 `day-*` / `evt` / `fact` 各自页面，**绝不在 _log 复述**。

（世界未创世 · 暂无操作记录。创世后每次操作在此记一行、新条置顶。）
