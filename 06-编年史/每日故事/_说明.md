# 每日故事 · 说明与模板

文件名：`第N天-故事.md`。每个周期结束时由「日结」动作（CLAUDE.md §4.7）自动生成的 **700–1500 字小说体**当期叙述。

> **名称是默认范例·单位随世界**："每日故事"作为温暖默认名保留（同 `day-`/`日结`，不全局改名）；史诗世界按 `周期量词` 读作「每纪故事」、文件落 `第N纪-故事.md`。详见宪法 §2.2「单位总注」。

- **只给用户读**：颗粒度比日志粗，**绝不进推进的任何读取层**——模拟器回溯一律走 `02-日志/`。
- 与编年史的关系：每日故事是"逐日细叙"，编年史是"跨期凝练回望"。
- 可回链当天日志与角色（单向；引擎不反向读本文件夹）。

模板：

```markdown
---
类型: 每日故事
周期: N
日期: ""
---

（700–1500 字小说体当日叙述。文学性、好读，不是流水账。可在文中或文末链回 [[day-第N天]]。）
```

## 文风（承 [[世界设定]]·文风）

写每日故事前对照 [[世界设定]] 的「文风 / 语言」段；**不参照最近几天的成稿**（防漂移，同 [[02-日志/_说明]] §格式铁律 #4）。

本节留给「每日故事」这一层的笔法细化（它是小说体、比日志松，可能要和日志不同的腔调）——随故事发展自填，暂留空。
